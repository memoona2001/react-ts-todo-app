# React TypeScript Todo App

A simple Todo application built with React, TypeScript, and Vite.

---

## 🚀 Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Clean and responsive UI
- Persistent state (if implemented)

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- CSS
